
# SyntaxWear - Tênis e Sneakers Online

Um e-commerce de calçados fictício, focado na venda de tênis e sneakers.

## Descrição

O SyntaxWear é um projeto de e-commerce que oferece uma ampla variedade de tênis e sneakers para todos os estilos e ocasiões. A plataforma foi desenvolvida com foco na experiência do usuário, proporcionando uma navegação intuitiva e um design moderno e atraente.

## Funcionalidades

- **Layout Responsivo:** O site se adapta a diferentes tamanhos de tela, como desktops, tablets e smartphones.
- **Navegação por Categorias:** Os produtos são organizados em categorias como Masculino, Feminino e Outlet.
- **Destaque de Produtos:** A página inicial apresenta os produtos em destaque e as principais categorias.
- **Rodapé Informativo:** O rodapé contém links úteis, informações de contato e um formulário de inscrição para newsletter.

## Tecnologias Utilizadas

- **HTML5:** Para a estruturação do conteúdo da página.
- **CSS3:** Para a estilização e o design do site, utilizando variáveis para um desenvolvimento mais eficiente e organizado.

## Estrutura de Arquivos

A estrutura de arquivos do projeto é organizada da seguinte forma:

```
/
├── css/
│   ├── base.css
│   ├── reset.css
│   ├── variables.css
│   └── components/
│       ├── footer.css
│       ├── header.css
│       ├── hero.css
│       ├── product-category.css
│       └── product-grid.css
├── images/
│   ├── banners/
│   ├── favicons/
│   ├── icons/
│   ├── logo/
│   └── products/
├── index.html
└── README.md
```

- **css/:** Contém todos os arquivos de estilo do projeto.
  - **components/:** Contém os arquivos de estilo para os componentes específicos do site.
- **images/:** Armazena todas as imagens utilizadas, como banners, ícones, logos e fotos de produtos.
- **index.html:** É o arquivo principal que estrutura a página inicial do e-commerce.
- **README.md:** Fornece informações detalhadas sobre o projeto.

## Como Utilizar

Para visualizar o site, basta abrir o arquivo `index.html` em qualquer navegador de sua preferência.

1. Clone o repositório:
   ```bash
   git clone https://github.com/Gabryela-Ribeiro/ecommerce-syntaxwear.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd ecommerce-syntaxwear
   ```
3. Abra o arquivo `index.html` no seu navegador.
